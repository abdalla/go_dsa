## DFS

### PRE ORDER
- ROOT > LEFT > RIGHT

![tree traversal](./images/traversal.jpg "tree traversal")

```
[A, B, D, E, C, F]
```

### IN ORDER
- LEFT > ROOT > RIGHT

![tree traversal](./images/traversal.jpg "tree traversal")

```
[D, B, E, A, F, C]
```

### POST ORDER
- LEFT > RIGHT > ROOT

![tree traversal](./images/traversal.jpg "tree traversal")

```
[D, E, B, F, C, A]
```
